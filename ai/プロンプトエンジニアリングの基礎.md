## Temperature
* この値が低ければ低い程、事実に基づいた出力が返る
* この値が高ければ高い程、ランダム性が増して創造的な出力が返る

|  | 応用例 |
| :-: | :-- |
| Temperature が低い | QA、レポート出力 |
| Temperature が高い | 詩、斬新なアイデア |
